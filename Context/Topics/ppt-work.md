# Topic: 장표 제작 및 번역

## 개요
전략추진실에서 가장 빈번한 업무. MPR, 사업계획, BCG 서포트, 타운홀, 링크온, 반상회, 이사회 등 다양한 장표를 제작하고 번역한다.

## 도구
- **제작**: Claude Code + Genspark 기반 (2026.03~ 전환 중)
- **번역**: Claude Code 기반 (용어집 적용)
- **폰트**: Pretendard (셰어포인트 > 8. 참고자료 > Pretendard 폰트)

## 장표 종류

| 장표 | 주기 | 특징 |
|------|------|------|
| MPR (Management Performance Review) | 반기 | 전사 성과 리뷰. 주주사 검토용 영문 버전 필요 |
| 사업계획 | 연간 (10~12월 집중) | 이사회/BoD 대응. 영문 동기화 필수 |
| BCG 서포트 | 프로젝트 기간 | 대규모 장표 작업. 엑셀 모델 기반 숫자 |
| 타운홀/링크온 | 비정기 | 전사 공유용 |
| 번역 장표 | BoD/주주 대응 시 | 한글 → 영문 |

## 제작 원칙
1. **One Page One Message** — 한 슬라이드에 하나의 메시지
2. **숫자 더블체크** — 데이터 연결(엑셀 링크) 금지, 값 직접 입력
3. **포맷 통일** — 폰트 Pretendard, 사이즈/여백/색상 기존 템플릿 준수
4. **아카이브** — 완료 후 PPTX + PDF 셰어포인트 저장

## 번역 원칙
- 용어집 기반 번역 (회사 고유 용어 일관성)
- Foundation → Fundamental 같은 명칭 변경 주의
- BoD 전날 긴급 요청 가능성 있음

## 파일 위치
- 셰어포인트 > 1. Deck File Archive (시계열별 정리)
- 완료 후 PPTX + PDF 모두 아카이브
