# Topic: Macro Analysis

## 개요
KOSIS(국가통계포털)에서 핵심 선후행 지표를 수집하여 월/분기/반기/연간 장표를 제작한다.

## 타임라인
- 월별 지표: 매월 말~익월 초 KOSIS 업데이트 확인
- 분기 장표: 분기 종료 후 2주 이내
- 장표 업데이트가 밀리면 일괄 업데이트 (예: 1~2월분 한꺼번에)

## 절차

### 1. KOSIS 데이터 수집
- KOSIS에서 지표별 최신 데이터 확인
- 주요 지표: 빈일자리, 채용인원, 근로자수, 입직자수, JK Revenue 등

### 2. Python 정리
- 수집한 데이터를 Python 스크립트로 통합 파일 생성
- (코드 경로 Phase 3에서 추가)

### 3. 장표 제작
- Claude Code + Genspark 기반으로 전환 중
- 셰어포인트 "4. Macro Analysis - 핵심 선후행 지표" 폴더에 저장

### 4. 지표 업데이트
- Macro Analysis Update: 신규 지표 발굴 (비정기)
- 창준님과 논의하여 추가/제거 결정

## 주의사항
- **KOSIS만 사용** (Metabase/태블로 미사용)
- 데이터 정확성 더블체크 필수 (고용탄성계수 등 계산 지표 특히 주의)
- 장표 그래프 단위 확인 (%, 명, 천명 등)

## 파일 위치
- 셰어포인트 > 4. Macro Analysis - 핵심 선후행 지표
