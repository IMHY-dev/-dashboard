# Topic: BCG 서포트

## 개요
BCG(보스턴컨설팅그룹) 프로젝트 진행 시 장표, 데이터, 엑셀 모델 등을 서포트한다.
2025.10~ 진행 중. 매번 요청이 다르므로 에이전트 자동화 대상이 아니며, 사람이 직접 판단한다.

## 특징
- 창준님이 슬랙으로 직접 요청 (대규모 장표 작업 — 한 번에 42개 스레드 수준)
- 엑셀 보안 해제 후 BCG팀 전달, 숫자 근거 Excel 정리 등
- BoD/이사회 연계 → 긴급 마감이 잦음
- Confidential 자료 다수 → 인쇄/공유 주의

## 에이전트 역할
- 슬랙에서 BCG 관련 요청 감지 시 "이런 요청이 왔습니다" 알림
- 관련 Context (Active Projects, 이전 BCG 작업 이력) 로드
- 실제 작업은 사람이 수행

## 파일 위치
