# Topic: 예산·구매 품의

## 개요
전략추진실의 모든 비용 집행은 예산품의 → 구매품의 → Concur 순서를 따른다.
예산품의는 "이만큼 쓰겠다"는 승인, 구매품의는 "이 업체에서 산다"는 승인.

## 프로세스

### 예산품의
1. 잡코리아 포탈 > 전자결재 > 문건등록
2. 문서분류 선택:
   - 1,000만원 이하: [예산관리] 1.예산관리 - 3)예산배정(신규, 전용, 초과) - 1,000만원 이하
   - 1,000만원 초과: [예산관리] 1.예산관리 - 3)예산배정(신규, 전용, 초과) - 1,000만원 초과
3. 첨부파일: 견적서, 이전 아웃풋(필요 시)
4. 읽기권한: 결재권자 + 동일팀
5. 코스트센터/G/L 계정은 업무별로 다름 → 모르면 경영기획팀 임장식님 또는 예산관리 Chatbot

### 구매품의
1. 예산품의 승인 완료 후, 해당 페이지에서 "하위문건 등록" 클릭
2. 문서분류: [구매] 해당 금액 구간 선택
3. 첨부파일: 견적서, 계약서(구매검토 완료분), 입찰생략요청서(500만원 이상 시)
4. 예산품의 읽기권한 없을 경우:
   - 방법 1: 경영기획팀에 열람권한 요청 → 하위문건으로 등록
   - 방법 2: 예산품의 PDF 첨부 + 비고란에 "예산품의 열람권한 없음에 따라 PDF 별첨" 코멘트

### 추가 예산 품의
- 반려 확률이 높으므로 증액 목적을 명확히 작성
- 잡코리아 포탈 > 전자결재 > 문건등록

## 주의사항
- 예산품의 없는 구매품의는 창준님께 먼저 보고
- 추가 예산 품의는 반려 확률 높음 → 명확한 증액 사유 필수
- 구매검토가 필요한 경우 (산학협력 등): 총무팀 이민희님에게 메일 (참조: 박현수, 남영현)

## 관련 인물
| 이름 | 역할 |
|------|------|
| 경영기획팀 임장식 | 예산 A to Z |
| 재무회계팀 박은미/왕윤형/최홍근 | 품의 세부 항목, 계정 가이드 |
| 총무팀 이민희 | 구매검토 (계약서 검토) |
| 총무팀 남영현 | 계약 담당 |
