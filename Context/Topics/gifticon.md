# Topic: 기프티콘 발송

## 개요
전략실 업무(산학협력, 인터뷰 리워드 등) 또는 ATS 운영팀 요청으로 기프티콘/네이버페이를 발송한다.

## 트리거
- 창준님이 기프티콘 발송 요청
- 산학협력프로젝트 리워드 지급
- ATS 운영팀 최돈민님이 고객 인터뷰 리워드 요청

## 프로세스

### Step 0. 사전 준비
1. 전달 대상자 리스트 수령 (이름 + 전화번호)
2. KT알파에 '거래고객코드 담당자' 변경 요청
   - b2b.m@kt.com (KT alpha G커머스운영팀 이덕희님)
   - kang.sh@kt.com (cc: 강석현 대리님)
   - 거래고객코드: C000162772 (전략추진실 고유코드)

### Step 1. 견적서 수령
- KT알파 강석현 대리님에게 견적서 요청 메일

### Step 2. 발주
- 견적서 확인 후 발주 요청 메일

### Step 3. 발송
- 대상자 리스트 전달 → KT알파에서 발송

### Step 4. Concur 처리
- 정산명세서 기반 Concur 송장 작성 (concur.md 참조)

## ATS 기프티콘 (별도)
- ATS 운영팀 최돈민님이 슬랙으로 요청
- 프로세스는 전략실 기프티콘과 동일
- 인턴은 전임자에 의해 최돈민님 포함 슬랙 DM방에 초대받아야 함

## 주의사항
- 아웃룩 메일로 진행해야 함 (에이전트 직접 처리 불가)
- 담당자 변경 요청을 먼저 해야 발주 가능
- 발송 후 정산명세서 수령까지 시간 소요
